# Notas & Ritmo

## Notas

```lilypond
c d e f g a b
```

## Notas Simultâneas

```lilypond
<c e g>
```

## Oitavas

```lilypond
c     % padrão
c'    % oitava acima
c''   % duas oitavas acima
c,    % oitava abaixo
c,,   % duas oitavas abaixo
```

## Duração

```lilypond
c1   % semibreve
e2   % mínima
d4   % semínima
g8   % colcheia
f16  % semicolcheia
a32  % fusa
b64  % semifusa
```

## Notas Pontuadas

```lilypond
c4.  % semínima pontuada
d8.  % colcheia pontuada
```
